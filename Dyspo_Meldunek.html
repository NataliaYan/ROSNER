Для эффективного планирования работы предусмотрен утренний отчет (мелдунек поранний)

Отчет предоставляется в виде краткого сообщения на ТХ и является обязательным.

Высылать его необходимо перед выездом.

Работа с диспонентом

Ежедневные отчеты и по понедельникам о паузе

Водитель обязан писать каждое утро перед выездом отчет (мельдунек). Шаблон отчета:

  • Witam / Здравствуйте /
  • Pauza / Пауза /
  • Do klienta – 200 km / До клиента 200 км /
  • Paliwa 70% / Топлива 70% /

Если был сделан виртуальный съезд на выходные, то напишите

  • Wirt zjazd-start o 3:00 (noc) / виртуальный съезд–старт в 3:00 (ночь) /

Если смена была закрыта поздно и водителю предстоит более поздний старт, то напишите об этом вечером

  • Koniec pracy o 23:30 / Конец работы в 23:30 /
  • Do klienta – 200 km / До клиента 200 км /
  • Paliwa 70% / Топлива 70% /
  • Planowana pauza – 11 g / Планируемая пауза 11 ч /
  • Start o 10:30 / Старт в 10:30 /
