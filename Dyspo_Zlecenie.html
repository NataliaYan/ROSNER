Работа с диспонентом

Обслуживание заказов (злецений) на ТХ - ocновные моменты:

В начале каждого дня и при работе с новым злецением необходимо сделать / контроль при выезде /

Для начала работы с новым злецением необходимо / подтвердить начало поездки /

Пуск > начало поездки > подтвердить

Открываем и читаем само злецение / BELADUNG /

адрес загрузки , номер загрузки и после слова Bem: дополнительная важная информация, касающаяся заказа, после прочтения - / ПУСК /

по прибытию на загрузку необходимо отмечать статусы:

    / прибытие под погрузку / --> / начало погрузки /--> / конец погрузки / --> / выезд с погрузки / (ответить на вопросы)

    Таким же способом читаем,едем и отмечаем статусы по разгрузке / ENTLADUNG /

    В конце работы со злецением необходимо закрыть / конец поездки /
